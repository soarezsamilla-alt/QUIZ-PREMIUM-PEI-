# **App Name**: PEI Facilitator

## Core Features:

- Interactive PEI Quiz: Guides teachers through a series of questions to understand their specific needs regarding Individualized Educational Plans (PEI).
- Dynamic Quiz Progress Display: Visualizes user progress through the quiz steps using progress bars and dot indicators.
- Personalized PEI Model Recommendation Tool: Utilizes AI to analyze quiz responses and recommend the most relevant PEI models, activities, or resources from the product catalog, tailoring suggestions to the teacher's profile and needs.
- Comprehensive PEI Resources Catalog: Displays various ready-to-use PEI models, supplemental activities, and exclusive bonuses in a well-organized sales page format.
- Real-time Limited-Time Offer Countdown: Shows a dynamic countdown timer for a special promotional price, creating urgency for purchase.
- Interactive FAQ Section: Provides an accordion-style frequently asked questions section to address common teacher inquiries efficiently.
- Secure Purchase Flow Information: Highlights security features and trust badges for a safe and confident buying experience.

## Style Guidelines:

- Primary color: Confident Rose (#BA2D5E), chosen for its engaging yet professional feel, supporting the educational content with warmth.
- Background color: Subtle Pale Rose (#F8F3F5), providing a clean, light canvas that is visually harmonious with the primary color.
- Accent color: Dynamic Lilac (#7A4DCC), selected to create vibrant highlights and calls to action, drawing the eye with its complementary hue.
- Headlines font: 'Playfair Display' (serif), adding an elegant and established touch for titles and key statements. Note: currently only Google Fonts are supported.
- Body font: 'DM Sans' (sans-serif), offering a modern, clear, and highly legible text for all descriptive content and quiz options. Note: currently only Google Fonts are supported.
- Leverage a consistent style of illustrative emojis and clear geometric symbols (e.g., checkmarks, arrows) to convey meaning concisely and maintain an approachable user experience across both quiz and sales sections.
- Employ a responsive, mobile-first design philosophy with content structured in digestible blocks within max-width containers, ensuring readability and ease of navigation on all device sizes.
- Incorporate subtle, functional animations such as 'slideUp' for card entrances and dynamic transform effects on interactive elements like buttons and options, providing smooth transitions and engaging feedback.
